# Seed 62 Artifacts

This folder contains the seed 62 experiment artifacts.

```text
seed_62/
  Train_Noise_Data/
  Gold_Evaluators/
  Experiments_Results/
  Supplemental_Ablation_Results/
  Noise_Baseline/
  docs/
  run_all_experiments.sh
```

Run all seed 62 experiments from the repository root:

```bash
bash seed_62/run_all_experiments.sh
```

Gold-guided and sweep results are stored directly by dataset in:

```text
seed_62/Experiments_Results/
```
