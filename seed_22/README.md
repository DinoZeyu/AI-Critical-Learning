# Seed 22 Artifacts

This folder contains the seed 22 experiment artifacts.

```text
seed_22/
  Train_Noise_Data/
  Gold_Evaluators/
  Experiments_Results/
  Supplemental_Ablation_Results/
  Noise_Baseline/
```

```bash

```

Gold-guided and sweep results are stored directly by dataset in:

```text
seed_22/Experiments_Results/
```
